# 情侣真心话大冒险 (Couple's Truth or Dare)

这是一款专为情侣设计的亲密互动网页应用。通过三个渐进的亲密等级（温和、进阶、情趣），结合真心话与大冒险的游戏形式，帮助情侣增进感情、探索彼此的内心世界与身体秘密。

## ✨ 主要功能

### 1. 💖 三个亲密等级
游戏设有三个递进的级别，适合不同阶段的氛围：
- **🌿 温和 (Gentle)**：适合破冰、回忆甜蜜往事，增进基础了解。
- **🔥 进阶 (Warming)**：氛围升温，涉及更深层的欲望讨论和轻度的身体接触。
- **💋 情趣 (Intimate)**：高强度互动，包含大胆的性幻想交流和角色扮演挑战（建议在私密空间进行）。

### 2. 🎭 角色扮演与个性化题目
- **角色系统**：玩家需设置昵称并选择 **Top (主导方)** 或 **Bottom (顺从方)** 角色。
- **智能题目适配**：系统会自动根据玩家的角色替换题目中的称呼（例如：`{{topPlayer}}` 命令 `{{bottomPlayer}}` 做某事），让挑战更具代入感。

### 3. 🎨 沉浸式体验
- **动态背景**：背景粒子效果会随着关卡切换而改变颜色和律动，营造不同的氛围。
- **倒计时挑战**：部分大冒险包含限时任务，内置计时器功能。
- **流畅动画**：使用 Framer Motion 打造丝滑的卡牌抽取和界面切换效果。

### 4. 🛠️ 自定义题库（隐藏功能）
应用内置了强大的题目编辑器，数据保存在浏览器的 LocalStorage 中。
- **开启方式**：在游戏首页或底部，**连续点击 5 次** 页脚的 "Created with love" 文字即可打开编辑器。
- **功能**：
  - 查看、编辑、删除现有题目。
  - 添加新题目（支持使用 `{{player1}}`, `{{player2}}`, `{{topPlayer}}`, `{{bottomPlayer}}` 占位符）。
  - **导入/导出**：支持将题库导出为 JSON 文件分享给朋友，或导入他人的题库。

## 📦 技术栈

- **React 18**：构建用户界面。
- **TypeScript**：提供类型安全。
- **Tailwind CSS**：快速构建现代化的样式。
- **Framer Motion**：处理复杂的动画交互。
- **LocalStorage**：本地持久化存储题目数据，无需联网即可使用。

## 🚀 如何开始

1. **设置玩家信息**：
   - 输入两人的名字。
   - 协商并选择各自的属性（Top/Bottom），注意两人不能选择相同的属性。
2. **选择关卡**：
   - 建议从“温和”开始，随着游戏进行，系统会在一定回合后提示升级到下一阶段。
3. **开始游戏**：
   - 轮流选择“真心话”或“大冒险”。
   - 按照卡牌指示完成任务。
   - 如果遇到限时任务，点击卡牌下方的计时器。

## 🔒 隐私说明

本项目是一个纯前端应用。所有的游戏数据（题目、设置）均存储在您浏览器的 **LocalStorage** 中，不会上传到任何服务器，请放心使用。
